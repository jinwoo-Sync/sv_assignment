## 6. 단위 테스트 결과 (구현 완료)

### 6.1 Logger (`sv_logger`)
- **싱글톤 검증**: `LoggerFactory`를 통한 유일 객체 보장 확인.
- **로그 레벨**: 설정된 레벨(`WARN` 등)에 따른 출력 필터링 확인.
- **안정성**: `LOG_XXXX` 매크로 연속 호출 시 크래시 없음 확인.

### 6.2 Memory Pool (`sv_core`)
- **기본 할당/해제**: 고정 크기 블록의 정상 순환 확인.
- **경계 검사**: 풀 용량 초과 시 에러 처리 및 외부 포인터 해제 시도 무시 확인.
